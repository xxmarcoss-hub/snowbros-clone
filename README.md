# Snow Bros Clone

Clone del classico gioco arcade Snow Bros (1990) realizzato in HTML5/Canvas + JavaScript.

## Caratteristiche

- 10 livelli + boss finale
- Multiplayer locale (2 giocatori)
- Sprite pixel art generati via codice
- Audio via Web Audio API
- Controlli da tastiera

## Controlli

### Giocatore 1 (Nick - Blu)
- **Frecce**: Movimento
- **Freccia Su**: Salto
- **Spazio**: Spara neve

### Giocatore 2 (Tom - Verde)
- **A/D**: Movimento
- **W**: Salto
- **Q**: Spara neve

### Generali
- **Enter**: Start/Pausa
- **1**: Modalità 1 giocatore
- **2**: Modalità 2 giocatori

## Come giocare

1. Apri `index.html` nel browser
2. Seleziona 1P o 2P
3. Copri i nemici di neve e spingili!
